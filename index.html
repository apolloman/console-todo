Console ToDo List

Simple Rules For start write your Own ToDo Tasks
  • 1. First open up the console in the Browser that you use.
  • 2. Then press the button.
  • 3. Before each action you should always enter a command from the list below.

Enter the following in the input field in the list below.
You need to always start with these commands!

  • "new" - Add a ToDo
  • "list" - View all Todos
  • "delete" - Remove todo item
  • "quit" - Quit app
